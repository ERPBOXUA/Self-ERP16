Cash flow analytic items.

This module adds the ability to specify analytics when conducting cash flow transactions. In particular, it adds and allows the user to generate a Cash Flow Statement in the context of these analytical items.

The module solves the basic problem of Odoo regarding:

  • the user's inability to specify cash flow items when conducting transactions with them and
  • the user's inability to generate a detailed Cash Flow Statement of the Company's financial and economic activities.

The module includes a standard directory of cash flow items (analytics).

Settings

Accounting / Configuration / Analytic Accounting / Analytic Plans

In the analytical plans to which the analytical accounts - cash flow items (CFI) will be assigned, the user should check the "Cash flow items" field.
If an analytical account belongs to an analytical plan that has a check mark in the "Cash flow item" field, it can be used as a CFI.

Usage

When posting a bank statement item, the user should specify the analytical account in the "Cash flow item" field. The analytical account will be used as an analytic for the bank and cash account (type "Bank and cash").

After the user clicks the "Validate" button, an accounting record is created in the system.
If there was a cash receipt, this record will have a "+" sign, if there was a write-off, it will have a "-" sign.

In order to analyze cash flows by item, the user needs to go to:

Accounting / Accounting / Analytic items

and apply the "Cash flow" filter

To generate a Cash Flow Statement with analytics by cash flow items, go to:

Accounting / Reporting / Cash Flow Analytic
