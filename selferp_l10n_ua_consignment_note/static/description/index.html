The module adds a printed form of the Consignment Note for Ukraine (Tovarno Transportna Nakladna).
The PDF report of Consignment Note for Ukraine is improved according to Ukrainian legislation requirements.

Usage

The printed form is available from the Stock (inventory) Transfer with the type OUT. The User should fill all necessary fields in the Additional info tab of the stock out transfer and click on the button "Consignment Note" to create the PDF report.
