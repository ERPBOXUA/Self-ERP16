Export VAT for Ukraine

The module adds the ability to create a VAT invoice from customer export invoice

This module extends the functionality of the module VAT accounting for Ukraine | Odoo Apps Store, which has the following features:

  • automatic calculation of the First VAT event for customers and vendors in terms of available 3 options: general for the counterparty, in the context of contracts, in the context of orders
  • automatic generation of the First VAT event journal entries
  • automatic generation of VAT invoices/adjustments
  • export of VAT invoices/adjustments to an xml file for further use in external softwares (submission to the regulatory authorities)
  • automatic generation of VAT adjustments in case of refunds and returning goods
  • import of vendors VAT documents into the system from xml files (generated/registered in external softwares)
  • handy report for analyzing the First VAT event, VAT liabilities and VAT credit of the Company
  • VAT Tax return: automatic filling with the ability to edit data
  • export of the VAT tax return to an xml file for further use in external softwares (submission to the regulatory authorities)
